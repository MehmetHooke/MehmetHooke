## 📌 Featured Projects

### QuickCV — Mobile CV Builder

A mobile CV creation application developed with **React Native and Expo Router**.

The application sends user data to a backend service deployed on **Google Cloud Run**. Dynamic HTML templates are processed with **Playwright**, converted into PDF documents, and returned to the mobile application.

**Technologies:** React Native, Expo Router, Node.js, Express, Playwright, Docker, Google Cloud Run

- [GitHub Repository](https://github.com/MehmetHooke/quick-cv)
- [Google Play](https://play.google.com/store/apps/details?id=com.anonymous.quicklycv)

### PianoSense — Piano Practice Analysis Platform

A multi-role mobile application designed to help piano students analyze short practice recordings and track their progress over time.

Students can select an exercise, listen to the reference audio, record their performance with metronome and count-in support, and submit the recording for analysis. The result screen displays an overall score, pitch accuracy, timing accuracy, and note-level feedback.

The application also includes dedicated experiences for teachers and parents. Teachers can create classes, generate participation codes, and monitor student progress, while parents can connect to their child’s account and follow recent performances.

Recordings are uploaded to Firebase Storage and processed asynchronously through Firebase Functions and Google Cloud Tasks. The audio analysis itself is handled by a separate Cloud Run service, while analysis progress and results are tracked through Firestore.

**Technologies:** React Native, Expo Router, TypeScript, Expo Audio, Firebase Authentication, Firestore, Firebase Storage, Firebase Functions, Google Cloud Tasks, Google Cloud Run

- [GitHub Repository](https://github.com/MehmetHooke/PianoSense)
